# 💳 Loan Approval Prediction using Logistic Regression

## 📌 Overview

This project predicts whether a loan application will be approved or not based on applicant details such as income, credit history, and demographic information.

## 🧠 Model Used

* Logistic Regression (Binary Classification)

## ⚙️ Workflow

1. Data loading
2. Handling missing values
3. Encoding categorical variables
4. Train-test split
5. Model training using Logistic Regression
6. Prediction on test data
7. Model evaluation

## 📊 Features Used

* Gender
* Marital Status
* Dependents
* Education
* Self Employment
* Applicant Income
* Co-applicant Income
* Loan Amount
* Loan Term
* Credit History
* Property Area

## 📊 Evaluation Metrics

* Accuracy Score
* Confusion Matrix (if applied)

## 📈 Result

The model predicts loan approval status based on applicant details and demonstrates the effectiveness of logistic regression for classification tasks.

## 🚀 How to Run

* Open the notebook in Google Colab
* Run all cells sequentially

## 📁 Files

* `logistic-regression-classification.ipynb`

## 📚 Key Learnings

* Binary classification using logistic regression
* Importance of credit history in loan approval
* Data preprocessing and encoding techniques
